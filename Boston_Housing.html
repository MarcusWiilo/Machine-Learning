Observação de Atributos - Desenvolvendo um Modelo¶

Para lembrar, estamos utilizando trs atributos do conjunto de dados dos imveis de Boston: 'RM','LSTAT'e'PTRATIO'`. Para cada observao de dados (vizinhana):

  • 'RM' o nmero mdio de quartos entre os imveis na vizinhana.
  • 'LSTAT' a porcentagem de proprietrios na vizinhana considerados de "classe baixa" (proletariado).
  • 'PTRATIO' a razo de estudantes para professores nas escolas de ensino fundamental e mdio na vizinhana.

from sklearn.tree import DecisionTreeRegressor
from sklearn.grid_search import GridSearchCV
from sklearn.metrics import make_scorer

def fit_model(X, y):
    """Desempenhar busca em matriz sobre o parmetro 'max_depth' para uma
    rvore de deciso de regresso treinada nos dados de entrada [x, y]"""
    
    # Gerar conjuntos de validao-cruzada para o treinamento de dados
    cv_sets = ShuffleSplit(X.shape[0], n_iter = 10, test_size = 0.20, random_state = 0)
    
    # TODO: Gerar uma rvore de deciso de regresso de objeto
    regressor = DecisionTreeRegressor(random_state=0)
    
    # TODO: Gerar um dicionrio para o parmetro 'max_depth' com um alcance de 1 a 10
    params = {'max_depth': range(1,11)}
    
    # TODO: Transformar 'performance_metric' em uma funo de pontuao utilizando 'make_scorer' 
    scoring_fnc = make_scorer(performance_metric)
    
     # TODO: Gerar o objeto de busca em matriz
    grid = GridSearchCV(regressor, params, scoring_fnc, cv=cv_sets, verbose=0)
    
    # Ajustar o objeto de busca em matriz com os dados para calcular o modelo timo
    grid = grid.fit(X, y)
    
    # Devolver o modelo timo depois de realizar o ajuste dos dados
    return grid.best_estimator_

Conclusão - Aplicabilidade¶

Caso exista um melhor treinamento, e assertividade nos testes, é possivel usar as configurações no mundo real. Os dados de 1978 podem ser usados como parametros de testes, já que auxiliam no momento de calibrar o algoritimo de aprendizagem. Os atributos presentes conseguem sim descrever um imóvel, porém poderia ser construido um modelo um pouco mais robusto. Sim, atravez da validção cruzada é possivel melhorar o treinamento e assertividade dos dados. Esses dados não serviriam para uma cidade rural, já que as variaveis como como nivel de pobreza da vizinhança e Razão estudante: professor das escolas proximas vai ser muito diferente do que em um ambiente urbano.¶
